MiceMapper is a web app that automates the processing and visualization of raw data from mice activity in a research environment. Users can upload CSV files, and the app dumps the data into a database, which can be accessed in an organized manner through tables and filters. The app aims to save researchers time and effort by automating the data organization process and facilitating data analysis.
